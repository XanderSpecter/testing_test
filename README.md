# testing_test

### The place to test some tests

## Запуск

-   Установить все зависимости:

```bash
npm ci
```

-   Запустить dev сервер:

```bash
npm run dev
```

-   Если всё прошло хорошо, сервер запустится по адресу [http://localhost:3000](http://localhost:3000)
